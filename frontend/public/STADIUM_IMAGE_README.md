# Stadium Background Image

Please add your stadium background image here named `stadium-background.jpg`.

You can use the uploaded image or any other stadium image with a sunset/dramatic sky.

Recommended specifications:
- Format: JPG or PNG
- Resolution: 1920x1080 or higher
- Aspect Ratio: 16:9
- File size: < 2MB for optimal loading

The image will be displayed with:
- brightness(0.55)
- saturate(0.95)  
- contrast(0.95)

To add the image:
1. Place `stadium-background.jpg` in the `/frontend/public/` folder
2. The component will automatically use it
